# Telégrafo — estación de práctica CW

Esta mini app está pensada para practicar Morse de forma sencilla, visual y progresiva desde un navegador, ya sea para aprenderlo desde cero o para reforzarlo de manera más dinámica.

Está construida como un único archivo HTML, por lo que se puede abrir directamente en el navegador o probarla online en Netlify en https://codigo-morse-online.netlify.app/.

## Qué ofrece

- Práctica libre con llave virtual: mantener presionado para emitir punto o raya.
- Medidor visual en tiempo real para distinguir el ritmo de cada señal.
- Cinta de texto con traducción automática de las letras emitidas.
- Reproducción de palabras escritas en Morse con audio.
- Abecedario interactivo para escuchar cada letra.
- Módulos de aprendizaje y entrenamiento con enfoque progresivo.
- Ejercicios de recuperación de memoria y práctica de tráfico radio.
- Seguimiento de progreso, rachas y precisión.
- Ajustes de tono, velocidad, volumen, espaciado Farnsworth y vibración.

## Cómo usarla

1. Puedes probar la versión online en https://codigo-morse-online.netlify.app/.
2. O abre [telegrafo.html](telegrafo.html) en tu navegador si prefieres usarla localmente.
3. Entra en la sección “Telégrafo” y mantén presionado el botón central o la lámpara para emitir Morse.
4. Usa la barra espaciadora si prefieres teclado.
5. Explora los demás apartados para aprender, entrenar, recuperar memoria y practicar señales radiofónicas.
6. Ajusta velocidad, tono y volumen desde la sección “Ajustes”.

## Secciones disponibles

- Historia: presentación de la estación y contexto del proyecto.
- Telégrafo: práctica libre con emisión y traducción en vivo.
- Aprender: ejercicio guiado basado en el método Koch.
- Entrenar: modo de copia y transmisión para trabajar precisión y reacción.
- Recuperar: repaso adaptativo de letras que cuestan más.
- Radio: ejercicios con tráfico CW realista y glosario de términos.
- Progreso: métricas de avance, precisión y logros.
- Ajustes: personalización del entorno de práctica.

## Controles rápidos

- Mantener presionado / barra espaciadora: emitir.
- Retroceso: borrar el último carácter.
- Limpiar: reiniciar la cinta.
- “Sonar”: reproducir la palabra escrita en Morse.
- Tocar una letra del abecedario: escuchar su código Morse.

## Notas de uso

- Está pensada para uso móvil y escritorio, con una interfaz táctil y sencilla.
- El audio depende del navegador y se activa al primer gesto del usuario.
- No requiere instalación ni dependencias externas.
